## Breaking vs Non-Breaking Changes

### Breaking Changes (Major Version)

<!-- FILL: Document what constitutes breaking changes -->
- Removing an endpoint
- Removing a field from response
- Changing field type
- Changing authentication mechanism
- Removing a query parameter

### Non-Breaking Changes (Minor/Patch)

- Adding a new endpoint
- Adding optional fields to request
- Adding fields to response
- Adding optional query parameters
- Bug fixes (patch)

## Error Responses

### Version Not Found

```json
{
  "error": {
    "code": "VERSION_NOT_SUPPORTED",
    "message": "API version v0 is not supported",
    "supported_versions": ["v1", "v2"],
    "documentation": "https://docs.example.com/api-versions"
  }
}
```

### Deprecated Version Warning

```json
{
  "data": {...},

      "message": "API v1 is deprecated and will be sunset on 2026-12-31",

}
```
