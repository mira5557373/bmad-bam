## Logging Standards

### Log Format (JSON)

```json
{
  "timestamp": "2026-04-28T12:00:00Z",
  "level": "INFO",
  "message": "Request processed",
  "tenant_id": "tenant_123",
  "trace_id": "abc123",
  "span_id": "def456",
  "module": "billing",
  "user_id": "user_789",
  "duration_ms": 45
}
```

### Log Levels

| Level | Usage | Tenant Visibility |
|-------|-------|-------------------|
| ERROR | Failures requiring attention | Internal only |
| WARN | Degraded but functional | Internal only |
| INFO | Normal operations | Internal only |
| DEBUG | Development/troubleshooting | Never in prod |
