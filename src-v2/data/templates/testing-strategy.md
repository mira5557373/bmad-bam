## AI Agent Testing

<!-- FILL: Document AI/agent testing approach -->
### Agent Test Types

| Type | Purpose | Approach |
|------|---------|----------|
| Tool unit tests | Verify tool logic | Mock LLM |
| Agent integration | Verify orchestration | Recorded responses |
| Agent E2E | Full flow | Real LLM (staging) |

### Test Scenarios

| Scenario | Setup | Verification |
|----------|-------|--------------|
| Successful run | Valid input + context | Output matches expected |
| Tool failure | Inject tool error | Graceful degradation |
| Token limit | Large context | Proper truncation |
| Tenant context | Multi-tenant input | Isolation maintained |

### LLM Mocking Strategy

```javascript
// Example: Mock LLM for deterministic testing
const mockLLM = {
  generate: jest.fn().mockResolvedValue({
    content: "Expected response",
    toolCalls: [{ name: "tool_a", args: {...} }]
  })
};
```
