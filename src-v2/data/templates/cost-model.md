## Tenant Cost Attribution

<!-- FILL: Document cost attribution approach -->
| Cost Type | Attribution Method | Accuracy |
|-----------|-------------------|----------|
| Direct (compute) | Metered usage | High |
| Shared (platform) | Proportional to usage | Medium |
| Fixed (compliance) | Even split | Low |

### Attribution Formula

```
Tenant Monthly Cost = 
  Direct Costs (metered)
  + (Shared Costs × Usage Proportion)
  + (Fixed Costs ÷ Active Tenants)
```
