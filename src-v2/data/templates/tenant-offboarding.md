## Offboarding Workflow

<!-- FILL: Document offboarding steps -->
```
1. Offboarding Initiated
       │
       ▼
2. Grace Period Started
       │
       ├── Downgrade to read-only
       ├── Notification emails sent
       │
       ▼
3. Data Export Window (30 days)
       │
       ├── Self-service export available
       ├── Daily reminder emails
       │
       ▼
4. Account Suspended
       │
       ├── All access revoked
       ├── API keys invalidated
       │
       ▼
5. Data Retention Period
       │
       ├── Data archived (per policy)
       │
       ▼
6. Data Purge
       │
       ├── PII deleted
       ├── Anonymized data retained (analytics)
       │
       ▼
7. Offboarding Complete
```
