### Plugin Manifest

```yaml
plugin_manifest:
  id: unique_plugin_id
  name: string
  version: semver
  author: string
  
  permissions:
    - data:read
    - ui:render
    - api:call
    
  tenant_scoping:
    isolation: strict  # Plugin runs in tenant context
    data_access: own_tenant_only
    
  resources:
    cpu_limit: 100m
    memory_limit: 128Mi
    timeout: 30s
```

### White-Label Schema

```yaml
white_label_config:
  tenant_id: uuid
  tier: enum[pro, enterprise, reseller]
  
  branding:
    logo_url: string
    favicon_url: string
    colors:
      primary: "#hex"
      secondary: "#hex"
      
  domain:
    custom_domain: string
    ssl_certificate: managed | byoc
    
  email:
    from_address: string
    reply_to: string
    templates: custom | default
    
  features:
    powered_by: show | hide
    custom_terms: string | null
```

### Reseller Architecture

```
┌─────────────────────────────────────────────────────────────────┐
│                    Reseller Hierarchy                            │
│                                                                  │
│  Platform (Root)                                                │
│       │                                                         │
│       ├── Reseller A (Tenant Type: reseller)                   │
│       │       ├── Customer A1 (Sub-tenant)                     │
│       │       ├── Customer A2 (Sub-tenant)                     │
│       │       └── Customer A3 (Sub-tenant)                     │
│       │                                                         │
│       └── Reseller B (Tenant Type: reseller)                   │
│               ├── Customer B1 (Sub-tenant)                     │
│               └── Customer B2 (Sub-tenant)                     │
│                                                                  │
│  Billing: Cascaded (Platform → Reseller → Customer)            │
│  Data: Isolated per tenant in hierarchy                        │
└─────────────────────────────────────────────────────────────────┘
```
