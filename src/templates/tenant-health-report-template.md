## Scoring Algorithm

### 3.1 Category Score Calculation

```
┌─────────────────────────────────────────────────────────────────┐
│                    Scoring Algorithm                             │
│                                                                  │
│  For each metric in category:                                    │
│                                                                  │
│    metric_score = normalize(actual_value, min, max, target)      │
│                                                                  │
│    where normalize():                                            │
│      if higher_is_better:                                        │
│        score = min(100, (value / target) * 100)                 │
│      else:                                                       │
│        score = max(0, (1 - (value / threshold)) * 100)          │
│                                                                  │
│  category_score = weighted_average(metric_scores, weights)       │
│                                                                  │
│  overall_score = weighted_average(category_scores, weights)      │
│                                                                  │
│  Adjustments:                                                    │
│    - Critical alerts: -10 points per active alert                │
│    - Recent incidents: -5 points per incident (last 30 days)     │
│    - Positive signals: +5 for expansion, +3 for advocacy         │
└─────────────────────────────────────────────────────────────────┘
```
