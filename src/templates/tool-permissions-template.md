## Contextual Permissions

### Runtime Conditions

```yaml
contextual_rules:
  - name: business_hours_only
    tools: [{{bh_tools}}]
    condition:
      time: "09:00-17:00"
      timezone: tenant_timezone
    action: allow
    fallback: deny_with_message
  
  - name: sensitive_data_access
    tools: [{{sensitive_tools}}]
    condition:
      data_classification: [confidential, restricted]
    action: require_approval
    approver: tenant_admin
  
  - name: high_cost_operations
    tools: [{{high_cost_tools}}]
    condition:
      estimated_cost: "> ${{cost_threshold}}"
    action: require_confirmation
    
  - name: cross_tenant_check
    tools: [all]
    condition:
      target_tenant: "!= current_tenant"
    action: deny
```
