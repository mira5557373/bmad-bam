## Redaction Methods

### Redaction Strategies

| Method             | Description            | Reversible             | Use Cases              |
| ------------------ | ---------------------- | ---------------------- | ---------------------- |
| Full Masking       | Replace with ***       | No                     | Logs, exports          |
| Partial Masking    | Show last 4 chars      | No                     | Display, receipts      |
| Tokenization       | Replace with token     | Yes (with key)         | Payment processing     |
| Encryption         | AES-256 encryption     | Yes (with key)         | Storage, transit       |
| Pseudonymization   | Replace with pseudo ID | Yes (with mapping)     | Analytics, testing     |
| Hashing            | One-way hash           | No                     | Deduplication          |
