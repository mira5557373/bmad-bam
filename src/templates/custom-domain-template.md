## Routing

### Routing Rules

| Priority           | Condition              | Target                 | Caching                |
| ------------------ | ---------------------- | ---------------------- | ---------------------- |
| 1                  | Host matches exact     | Tenant origin          | Per-tenant             |
| 2                  | Host matches wildcard  | Tenant resolution      | Short TTL              |
| 3                  | Path starts with /api  | API origin             | No cache               |
| 4                  | Path starts with /static| Static CDN            | Long TTL               |
| 5                  | Default                | Platform default       | Standard               |

### Tenant Resolution

| Method             | Input                  | Resolution             | Caching                |
| ------------------ | ---------------------- | ---------------------- | ---------------------- |
| Domain lookup      | Host header            | Domain -> Tenant ID    | {{domain_cache_ttl}}   |
| Subdomain parse    | Host header            | Subdomain -> Tenant ID | {{subdomain_cache_ttl}}|
| Path prefix        | URL path               | /t/{tenant}/ -> Tenant | No cache               |

### Routing Headers

| Header             | Direction              | Value                  | Purpose                |
| ------------------ | ---------------------- | ---------------------- | ---------------------- |
| X-Tenant-ID        | Edge -> Origin         | Resolved tenant ID     | Tenant context         |
| X-Original-Host    | Edge -> Origin         | Original host header   | Logging, redirects     |
| X-Forwarded-Host   | Edge -> Origin         | Original host          | Standard proxy header  |
| X-Custom-Domain    | Edge -> Origin         | true/false             | Custom domain flag     |

### Failover Configuration

| Scenario           | Detection              | Action                 | Recovery               |
| ------------------ | ---------------------- | ---------------------- | ---------------------- |
| Origin down        | Health check failure   | Serve cached or error  | Auto on health pass    |
| SSL failure        | Certificate error      | Redirect to platform   | On cert fix            |
| DNS issue          | Resolution failure     | Use IP fallback        | Auto on DNS fix        |
| Tenant suspended   | Config flag            | Serve suspension page  | On reactivation        |
