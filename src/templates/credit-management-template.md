```yaml

  balance_after: decimal(18,6)

## Credit Operations

### Purchase Flow

```
Customer → Payment Gateway → Webhook → Credit Service → Ledger → Balance Update
    │                                         │
    └── Confirmation ←── Receipt ←── Success ─┘
```

### Consumption Flow

```
API Request → Rate Limiter → Credit Check → Service Execution → Debit → Response
                   │              │                                │
                   └── Reject ←── Insufficient ←── Fail ───────────┘
