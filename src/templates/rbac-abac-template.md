### Policy Expression Examples

```
# Tenant isolation policy
permit if user.tenant_id == resource.owner_tenant_id

# Time-based access
permit if user.roles contains "admin" 
       AND environment.current_time.hour >= 9 
       AND environment.current_time.hour <= 17

# Clearance level check
permit if user.clearance_level >= resource.sensitivity

# Combined RBAC + ABAC
permit if user.roles contains "editor" 
       AND resource.classification != "restricted"
       AND user.department == resource.department
```
