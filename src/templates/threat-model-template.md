## Attack Trees

### Cross-Tenant Data Access

```
[Goal: Access Another Tenant's Data]
    │
    ├── [OR] Bypass RLS
    │   ├── [AND] SQL Injection + RLS Disabled
    │   ├── [AND] Direct DB Access + Missing Policies
    │   └── [AND] Admin Impersonation + Query Override
    │
    ├── [OR] API Exploitation
    │   ├── [AND] IDOR + Missing AuthZ Check
    │   ├── [AND] Batch API + Tenant ID Manipulation
    │   └── [AND] Cache Poisoning + Tenant Key Collision
    │
    └── [OR] Infrastructure Compromise
        ├── [AND] Container Escape + Shared Storage
        └── [AND] Network Sniffing + Unencrypted Traffic
```

### AI Agent Compromise

```
[Goal: Compromise AI Agent]
    │
    ├── [OR] Prompt Injection
    │   ├── [AND] Direct Input + No Validation
    │   ├── [AND] Document Injection + RAG Retrieval
    │   └── [AND] Tool Output + Agent Processing
    │
    ├── [OR] Tool Exploitation
    │   ├── [AND] Code Execution Tool + Escape Sandbox
    │   └── [AND] API Tool + Credential Access
    │
    └── [OR] Context Manipulation
        ├── [AND] History Injection + Continued Session
        └── [AND] System Prompt Override + Jailbreak
```
