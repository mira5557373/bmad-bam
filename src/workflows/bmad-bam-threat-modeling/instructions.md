# Threat Modeling

Design threat modeling processes for multi-tenant AI platforms. Use when the user requests to 'design threat model' or 'create threat assessment'.

## Quick Reference

- **Owner:** Security Architect
- **Domain:** security
- **Complexity:** complex
- **Headless:** Yes

## Workflow Triggers

- **architect-bam (Atlas)**: When planning secure architecture or evaluating attack surfaces -> `bam/workflows/threat-modeling`
