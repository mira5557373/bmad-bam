# Continuous Security Setup

Set up continuous security monitoring and automation. Use when the user requests to 'setup security monitoring' or 'configure continuous security'.

## Quick Reference

- **Owner:** Security Engineer (Atlas)
- **Domain:** security
- **Complexity:** complex
- **Headless:** Yes

## Workflow Triggers

| Trigger | Action |
|---------|--------|
| "setup security monitoring" | Start Create mode |
| "configure continuous security" | Start Create mode |
| "modify security setup" | Start Edit mode |
| "validate security setup" | Start Validate mode |
