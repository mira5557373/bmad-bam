# AI Observability Setup

Set up AI/LLM observability for multi-tenant platform. Use when the user requests to 'setup AI observability' or 'configure LLM monitoring'.

## Quick Reference

- **Owner:** AI Platform Engineer (Nova)
- **Domain:** ai-operations
- **Complexity:** complex
- **Headless:** Yes

## Workflow Triggers

| Trigger | Action |
|---------|--------|
| "setup AI observability" | Start Create mode |
| "configure LLM monitoring" | Start Create mode |
| "modify AI observability" | Start Edit mode |
| "validate AI observability" | Start Validate mode |
