# Product Analytics Instrumentation - Instructions

## Quick Start

1. Ensure master architecture exists
2. Run workflow in Create mode for new design
3. Follow step files sequentially
4. Output saved to `{output_folder}/planning-artifacts/analytics/`

## Key Decisions

- **Event Taxonomy**: Naming conventions and categories
- **Schema Design**: Required vs optional properties
- **Pipeline Architecture**: Real-time vs batch processing
- **Governance**: Who can add/modify events

## Common Customizations

- Add tenant-specific event properties
- Include AI/ML feature events
- Configure compliance-aware data handling
- Enable event versioning
