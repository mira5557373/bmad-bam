# AI Context Management

---

## Mode Selection

| Mode | Description | Entry Point |
|------|-------------|-------------|
| **Create** | Generate new AI Context Management from scratch | `steps/step-01-c-*` |
| **Edit** | Modify existing Context Management | `steps/step-10-e-*` |
| **Validate** | Check Context Management against criteria | `steps/step-20-v-*` |

### Create Mode
- `step-01-c-context-strategy-design.md` - Design token budget and prioritization
- `step-02-c-compression-techniques.md` - Design compression approaches
- `step-03-c-caching-retrieval.md` - Design caching and RAG integration

### Edit Mode
- `step-10-e-load-existing.md` - Load existing context management
- `step-11-e-apply-changes.md` - Apply modifications

### Validate Mode
- `step-20-v-load-artifact.md` - Load context management artifacts
- `step-21-v-validate.md` - Run validation checks
- `step-22-v-generate-report.md` - Generate validation report
