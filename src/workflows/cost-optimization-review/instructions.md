# Cost Optimization Review

Execute cost optimization reviews for multi-tenant AI platform. Use when the user requests to 'review costs' or 'optimize costs'.

## Quick Reference

- **Owner:** FinOps Engineer (Atlas)
- **Domain:** cost-management
- **Complexity:** standard
- **Headless:** Yes

## Workflow Triggers

| Trigger | Action |
|---------|--------|
| "review costs" | Start Create mode |
| "optimize costs" | Start Create mode |
| "modify cost review" | Start Edit mode |
| "validate cost review" | Start Validate mode |
