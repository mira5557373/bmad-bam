# Performance Review Execution

Execute periodic performance reviews for multi-tenant AI platform. Use when the user requests to 'review performance' or 'execute performance review'.

## Quick Reference

- **Owner:** Performance Engineer (Atlas)
- **Domain:** operations
- **Complexity:** standard
- **Headless:** Yes

## Workflow Triggers

| Trigger | Action |
|---------|--------|
| "review performance" | Start Create mode |
| "execute performance review" | Start Create mode |
| "modify performance review" | Start Edit mode |
| "validate performance review" | Start Validate mode |
