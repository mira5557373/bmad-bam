# Security Operations

Set up security operations including threat detection, correlation rules, AI threat detection, and hunting capabilities. Use for SOC/SIEM configuration.

## Quick Reference

- **Owner:** Security Operations Engineer
- **Domain:** security
- **Complexity:** complex
- **Headless:** Yes

## Workflow Triggers

- **security-bam**: When configuring threat detection or SIEM rules -> `bam/workflows/security-operations`
- **devops-bam**: When setting up security monitoring infrastructure -> `bam/workflows/security-operations`
