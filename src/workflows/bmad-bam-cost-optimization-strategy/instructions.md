# Cost Optimization Strategy - Instructions

## Overview

This workflow guides the design of cost optimization strategies for multi-tenant AI platforms. It covers cloud resource optimization, LLM cost reduction, tenant-aware allocation, and FinOps practices.

## Quick Start

1. Ensure master architecture is defined
2. Run cost optimization strategy workflow
3. Follow step-by-step guidance
4. Produce cost optimization documentation

## Workflow Steps

### Create Mode (Steps 01-04)
1. **Analyze Costs** - Examine current cost structure and drivers
2. **Design Optimization** - Create optimization strategies per category
3. **Design FinOps** - Establish FinOps practices and governance
4. **Assembly** - Compile final cost optimization strategy document

### Edit Mode (Steps 10-11)
1. **Load Existing** - Load current cost optimization strategy
2. **Apply Changes** - Make targeted modifications

### Validate Mode (Steps 20-22)
1. **Load Artifact** - Load strategy for validation
2. **Validate** - Run quality checks
3. **Report** - Generate validation report

## Key Deliverables

- Cost optimization strategy document
- FinOps recommendations
- Cost reduction roadmap
- Governance framework
