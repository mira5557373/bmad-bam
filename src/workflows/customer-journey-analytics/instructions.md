# Customer Journey Analytics - Instructions

## Quick Start

1. Ensure product analytics instrumentation exists
2. Run workflow in Create mode for new design
3. Follow step files sequentially
4. Output saved to `{output_folder}/planning-artifacts/analytics/`

## Key Decisions

- **Journey Definition**: Which journeys to track
- **Touchpoint Granularity**: Level of detail
- **Attribution Model**: How to assign credit
- **Optimization Goals**: What to improve

## Common Customizations

- Add persona-specific journeys
- Include cross-channel tracking
- Configure real-time journey visualization
- Enable predictive next-best-action
