# Incident Response Operations

Execute incident response procedures for production issues. Use when the user requests to 'handle incident' or 'initiate incident response'.

## Quick Reference

- **Owner:** Incident Commander (Atlas)
- **Domain:** operations
- **Complexity:** complex
- **Headless:** Yes

## Workflow Triggers

| Trigger | Action |
|---------|--------|
| "handle incident" | Start Create mode |
| "initiate incident response" | Start Create mode |
| "update incident" | Start Edit mode |
| "validate incident resolution" | Start Validate mode |
