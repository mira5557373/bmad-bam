# Service Mesh Configuration

Design service mesh for multi-tenant microservices. Use when the user requests to 'design service mesh' or 'configure mesh networking'.

## Quick Reference

- **Owner:** Platform Architect (Atlas)
- **Domain:** infrastructure, networking
- **Complexity:** complex
- **Headless:** Yes

## Workflow Triggers

- **architect-bam (Atlas)**: When designing service mesh architecture -> `bam/workflows/service-mesh-configuration`
- **devops-bam**: When implementing mesh configuration or traffic policies -> `bam/workflows/service-mesh-configuration`
