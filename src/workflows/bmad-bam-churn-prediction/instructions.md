# Churn Prediction - Instructions

## Quick Start

1. Ensure tenant health scoring design exists
2. Run workflow in Create mode for new design
3. Follow step files sequentially
4. Output saved to `{output_folder}/planning-artifacts/analytics/`

## Key Decisions

- **Indicator Selection**: Which signals predict churn
- **Model Architecture**: ML vs rule-based approach
- **Risk Thresholds**: What scores trigger intervention
- **Intervention Timing**: When to act on predictions

## Common Customizations

- Adjust indicators based on product type
- Add industry-specific churn signals
- Configure tier-specific intervention playbooks
- Enable real-time prediction scoring
