# Security Audit Execution

Execute quarterly security audits including access control review, vulnerability assessment, and compliance verification. Use for scheduled security assessments.

## Quick Reference

- **Owner:** Security Architect
- **Domain:** security
- **Complexity:** complex
- **Headless:** Yes

## Workflow Triggers

- **security-bam**: When executing scheduled security audits or compliance checks -> `bam/workflows/security-audit-execution`
- **devops-bam**: When running security scans or vulnerability assessments -> `bam/workflows/security-audit-execution`
