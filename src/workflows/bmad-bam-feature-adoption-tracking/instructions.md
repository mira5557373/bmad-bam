# Feature Adoption Tracking - Instructions

## Quick Start

1. Ensure product analytics instrumentation exists
2. Run workflow in Create mode for new design
3. Follow step files sequentially
4. Output saved to `{output_folder}/planning-artifacts/analytics/`

## Key Decisions

- **Feature Taxonomy**: How to categorize features
- **Adoption Definition**: What counts as "adopted"
- **Funnel Design**: Key activation milestones
- **Success Criteria**: Per-feature targets

## Common Customizations

- Add tier-specific feature tracking
- Include A/B test integration
- Configure per-feature SLOs
- Enable automated recommendations
