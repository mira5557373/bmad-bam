# Tenant Analytics Dashboard

Design analytics dashboards for multi-tenant SaaS platforms. Use when the user requests to 'design analytics' or 'create tenant dashboard'.

## Quick Reference

- **Owner:** Platform Architect (Atlas)
- **Domain:** tenant-lifecycle
- **Complexity:** intermediate
- **Headless:** Yes

## Workflow Triggers

- **architect-bam (Atlas)**: When designing tenant-facing analytics -> `bam/workflows/tenant-analytics-dashboard`
- **ux-bam**: When designing dashboard UX for tenant analytics -> `bam/workflows/tenant-analytics-dashboard`
