# RBAC/ABAC Design

Design role-based and attribute-based access control for multi-tenant platforms. Use when the user requests to 'design access control' or 'implement RBAC/ABAC'.

## Quick Reference

- **Owner:** Security Architect (Atlas)
- **Domain:** security
- **Complexity:** complex
- **Headless:** Yes

## Workflow Triggers

- **architect-bam (Atlas)**: When designing access control or authorization policies -> `bam/workflows/rbac-abac-design`
- **security-bam**: When implementing RBAC/ABAC or validating authorization -> `bam/workflows/rbac-abac-design`
