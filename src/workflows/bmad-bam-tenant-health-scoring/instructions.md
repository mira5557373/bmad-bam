# Tenant Health Scoring - Instructions

## Quick Start

1. Ensure tenant health monitoring design exists
2. Run workflow in Create mode for new design
3. Follow step files sequentially
4. Output saved to `{output_folder}/planning-artifacts/analytics/`

## Key Decisions

- **Metric Selection**: Which metrics contribute to health score
- **Weight Assignment**: Relative importance of each metric
- **Normalization**: How to scale different metric types
- **Thresholds**: What scores trigger what actions

## Common Customizations

- Adjust weights based on business priorities
- Add industry-specific metrics
- Configure tier-specific thresholds
- Enable predictive trending
