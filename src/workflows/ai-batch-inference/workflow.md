# AI Batch Inference

---

## Mode Selection

| Mode | Description | Entry Point |
|------|-------------|-------------|
| **Create** | Generate new AI Batch Inference from scratch | `steps/step-01-c-*` |
| **Edit** | Modify existing Batch Inference | `steps/step-10-e-*` |
| **Validate** | Check Batch Inference against criteria | `steps/step-20-v-*` |

### Create Mode
- `step-01-c-job-queue-design.md` - Design queue architecture
- `step-02-c-execution-engine-design.md` - Design parallel execution
- `step-03-c-result-handling-design.md` - Design result aggregation

### Edit Mode
- `step-10-e-load-existing.md` - Load existing batch inference
- `step-11-e-apply-changes.md` - Apply modifications

### Validate Mode
- `step-20-v-load-artifact.md` - Load batch inference artifacts
- `step-21-v-validate.md` - Run validation checks
- `step-22-v-generate-report.md` - Generate validation report
