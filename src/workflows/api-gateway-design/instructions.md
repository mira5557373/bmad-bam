# API Gateway Design

Design API gateway configuration with rate limiting, authentication, and routing. Use when the user requests to 'configure API gateway' or 'design gateway routing'.

## Quick Reference

- **Owner:** Platform Architect (Atlas)
- **Domain:** platform
- **Complexity:** medium
- **Headless:** Yes

## Workflow Triggers

- **architect-bam (Atlas)**: When designing API gateway architecture or routing patterns -> `bam/workflows/api-gateway-design`
- **devops-bam**: When implementing API gateway infrastructure or configuring Kong/AWS API Gateway -> `bam/workflows/api-gateway-design`
