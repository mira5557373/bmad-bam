# Cohort Analysis - Instructions

## Quick Start

1. Ensure tenant health scoring design exists
2. Run workflow in Create mode for new design
3. Follow step files sequentially
4. Output saved to `{output_folder}/planning-artifacts/analytics/`

## Key Decisions

- **Cohort Dimensions**: Time-based vs behavior-based
- **Retention Period**: How long to track cohorts
- **Segmentation**: Which attributes to segment by
- **Visualization**: Charts and dashboards

## Common Customizations

- Add acquisition channel cohorts
- Include tier-based segmentation
- Configure custom retention windows
- Enable automated insights
