# AI Feedback Loop

---

## Mode Selection

| Mode | Description | Entry Point |
|------|-------------|-------------|
| **Create** | Generate new AI Feedback Loop from scratch | `steps/step-01-c-*` |
| **Edit** | Modify existing Feedback Loop | `steps/step-10-e-*` |
| **Validate** | Check Feedback Loop against criteria | `steps/step-20-v-*` |

### Create Mode
- `step-01-c-feedback-collection-design.md` - Design feedback collection
- `step-02-c-feedback-analysis-pipeline.md` - Design analysis pipeline
- `step-03-c-model-improvement-integration.md` - Design improvement integration

### Edit Mode
- `step-10-e-load-existing.md` - Load existing feedback loop
- `step-11-e-apply-changes.md` - Apply modifications

### Validate Mode
- `step-20-v-load-artifact.md` - Load feedback loop artifacts
- `step-21-v-validate.md` - Run validation checks
- `step-22-v-generate-report.md` - Generate validation report
