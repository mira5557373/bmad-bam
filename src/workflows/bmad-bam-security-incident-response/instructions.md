# Security Incident Response

Design security-specific incident response procedures for multi-tenant AI platforms. Use when the user requests to 'design security incident response' or 'create security incident procedures'.

## Quick Reference

- **Owner:** Security Architect
- **Domain:** security
- **Complexity:** complex
- **Headless:** Yes

## Workflow Triggers

- **security-bam**: When designing security incident response procedures or breach notification workflows -> `bam/workflows/security-incident-response`
- **devops-bam**: When implementing security monitoring or incident automation -> `bam/workflows/security-incident-response`
