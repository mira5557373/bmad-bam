# Capacity Planning Review

Execute capacity planning reviews for growth and scaling. Use when the user requests to 'review capacity' or 'plan capacity'.

## Quick Reference

- **Owner:** Capacity Planning Engineer (Atlas)
- **Domain:** operations
- **Complexity:** standard
- **Headless:** Yes

## Workflow Triggers

| Trigger | Action |
|---------|--------|
| "review capacity" | Start Create mode |
| "plan capacity" | Start Create mode |
| "modify capacity plan" | Start Edit mode |
| "validate capacity plan" | Start Validate mode |
