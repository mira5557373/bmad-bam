| Critical Alerts        | > 0            | {{alert_current}} | {{rb_alert}}    |
| Tenant Complaints      | > 3            | {{complaint_current}} | {{rb_complaint}} |

### Decision Matrix

```
IF any critical test FAILED
    └── IMMEDIATE ROLLBACK required
        └── Execute rollback procedure (Section: Rollback Procedure)

ELSE IF error rate > threshold OR latency > threshold
    └── ESCALATE to on-call lead
        ├── IF issue isolated to single component
        │       └── Attempt hotfix (15 min window)
        └── ELSE
                └── ROLLBACK recommended

ELSE IF minor issues detected
    └── MONITOR for 30 minutes
        ├── IF issues persist
        │       └── ESCALATE
        └── ELSE
                └── PROCEED to sign-off

ELSE
    └── PROCEED to sign-off
```
