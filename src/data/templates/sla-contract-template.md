## Performance Metrics

### Metric Definitions

#### Monthly Uptime Percentage

The Monthly Uptime Percentage is calculated as:

```
((Total Minutes in Month - Downtime Minutes) / Total Minutes in Month) x 100
```

Where Downtime Minutes is the total number of minutes during which the Service was unavailable, excluding Scheduled Maintenance.

#### Error Rate

The Error Rate is calculated as:

```
(Number of Failed Requests / Total Requests) x 100
```

Where Failed Requests are requests returning HTTP 5xx status codes or timing out.
