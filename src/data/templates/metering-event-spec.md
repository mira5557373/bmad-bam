## Event Schema

### Base Event Structure

```json
{
  "event_id": "{{event_id}}",
  "event_type": "{{event_type}}",
  "timestamp": "{{timestamp}}",
  "tenant_id": "{{tenant_id}}",
  "user_id": "{{user_id}}",
  "tier": "{{tier}}",
  "source": "{{source_service}}",
  "idempotency_key": "{{idempotency_key}}",
  "payload": {}
}
```

### Required Fields

| Field           | Type       | Required   | Description        |
| --------------- | ---------- | ---------- | ------------------ |
| event_id        | string     | Yes        | Unique event ID    |
| event_type      | string     | Yes        | Event category     |
| timestamp       | ISO8601    | Yes        | Event time (UTC)   |
| tenant_id       | string     | Yes        | Tenant identifier  |
| idempotency_key | string     | Yes        | Deduplication key  |

```json
{
  "event_type": "usage.tokens",
  "payload": {
    "model": "{{model_name}}",

    "output_tokens": {{output_tokens}},
    "cached_tokens": {{cached_tokens}},
    "agent_id": "{{agent_id}}",
    "task_id": "{{task_id}}"
  }
}
```

### Agent Execution

```json
{
  "event_type": "usage.agent_run",
  "payload": {
    "agent_type": "{{agent_type}}",
    "duration_ms": {{duration_ms}},
    "tool_calls": {{tool_call_count}},
    "success": {{success}},
    "task_id": "{{task_id}}"
  }
}
```

### Tool Invocation

```json
{
  "event_type": "usage.tool_call",
```
