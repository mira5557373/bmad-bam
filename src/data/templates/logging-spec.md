## Tenant Context

### Required Fields

All log entries MUST include:

| Field       | Source              | Required |
| ----------- | ------------------- | -------- |
| tenant_id   | Request context     | Yes      |
| user_id     | Auth token          | Conditional |
| tier        | Tenant config       | Yes      |
| region      | Tenant config       | Yes      |

### Tier-Specific Logging

| Tier       | Log Detail     | Retention      | Export       |
| ---------- | -------------- | -------------- | ------------ |
| Free       | Standard       | 7 days         | None         |
| Pro        | Enhanced       | 30 days        | On request   |
| Enterprise | Full           | 90 days        | Real-time    |

## Structured Logging

### Event Categories

| Category         | Pattern                  | Example            |
| ---------------- | ------------------------ | ------------------ |
| Authentication   | auth.*                   | auth.login.success |
| Agent execution  | agent.*                  | agent.task.started |
| Tool invocation  | tool.*                   | tool.call.completed|
| Billing event    | billing.*                | billing.usage.recorded |

### Common Events

#### Agent Events

```json
{
  "event": "agent.task.completed",
  "tenant_id": "{{tenant_id}}",
  "agent_id": "{{agent_id}}",
  "task_id": "{{task_id}}",
  "duration_ms": {{duration}},
  "token_count": {{tokens}},
  "success": true
}
```

#### Tool Events

```json
{
  "event": "tool.call.completed",
  "tenant_id": "{{tenant_id}}",
  "tool_name": "{{tool_name}}",
  "duration_ms": {{duration}},
  "success": true
}
```
