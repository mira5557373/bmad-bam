## Endpoint Management

### Endpoint Configuration

```yaml
endpoint_config:
  url: https://example.com/webhook
  method: POST
  headers:
    Content-Type: application/json
    X-Custom-Header: {{custom_value}}
  authentication:
    type: {{auth_type}}  # hmac, bearer, basic, none
    secret: ${WEBHOOK_SECRET}
  events:
    - agent.completed
    - task.created
  filters:
    - field: resource_type
      operator: equals
      value: "critical"
```

### Endpoint Validation

| Validation         | Requirement            | Action on Failure      |
| ------------------ | ---------------------- | ---------------------- |
| URL Format         | Valid HTTPS URL        | Reject configuration   |
| SSL Certificate    | Valid, not expired     | Warn or reject         |
| Connectivity       | Reachable              | Warn                   |
| Response           | 2xx within timeout     | Mark unhealthy         |

### Endpoint Health

| Degraded           | {{degraded_fail}}% failure | Increased monitoring |

### Signature Generation

```
payload = JSON.stringify(event)
```
