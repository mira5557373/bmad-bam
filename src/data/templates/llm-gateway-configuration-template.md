### Cache Key Strategy

| Strategy | Key Components | Use Case |
|----------|----------------|----------|
| Exact Match | hash(prompt + model + params) | Repeated queries |
| Semantic | embedding(prompt) + threshold | Similar queries |
| Prefix | hash(system_prompt) + partial | Shared context |
| Tenant-Scoped | tenant_id + hash(prompt) | Multi-tenant |

### Cache Invalidation

| Trigger | Action | Scope |
|---------|--------|-------|
| Model Update | Invalidate model cache | Per model |
| Prompt Change | Invalidate prompt cache | Per prompt template |
| TTL Expiry | Auto-evict | Per entry |
| Manual Purge | Invalidate all | Global/Tenant |
| Cost Threshold | Reduce cache size | Global |
