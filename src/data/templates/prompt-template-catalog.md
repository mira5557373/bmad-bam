## Multi-Tenant Customization

### Customization Levels

| Level        | Scope            | Who Can Modify       | Examples                |
| ------------ | ---------------- | -------------------- | ----------------------- |
| Platform     | All tenants      | Platform admins      | Core system prompts     |
| Tier         | Tenants in tier  | Platform admins      | Tier-specific features  |
| Tenant       | Single tenant    | Tenant admins        | Brand voice, domain     |
| User         | Single user      | User (if allowed)    | Personal preferences    |

### Tenant-Specific Overrides

```yaml
tenant_customization:
  tenant_id: {{tenant_id}}
  tenant_name: {{tenant_name}}
  tier: {{tenant_tier}}
  
  prompt_overrides:
    - prompt_id: {{prompt_id}}
      override_type: append
      content: |
        Additional tenant-specific instructions:
        - {{tenant_instruction_1}}
        - {{tenant_instruction_2}}
    
    - prompt_id: {{prompt_id_2}}
      override_type: replace_section
      section: "brand_voice"
      content: |
        {{tenant_brand_voice}}
  
  variable_defaults:
    company_name: "{{tenant_company}}"
    industry: "{{tenant_industry}}"
    tone: "{{tenant_tone}}"
  
  model_overrides:
    preferred_model: {{tenant_model}}
    temperature: {{tenant_temp}}
```

### Override Types

| Type            | Description                               | Use Case                    |
| --------------- | ----------------------------------------- | --------------------------- |
| append          | Add content to end of prompt              | Additional instructions     |
| prepend         | Add content to start of prompt            | Context setting             |
| replace_section | Replace named section                     | Brand voice customization   |
| replace_all     | Complete prompt replacement               | Fully custom prompts        |
| variable_inject | Inject tenant-specific variables          | Company name, domain terms  |

### Tier-Based Features

| Feature                  | FREE | PRO | ENTERPRISE |
| ------------------------ | ---- | --- | ---------- |
| Custom brand voice       | No   | Yes | Yes        |
| Custom prompts           | No   | No  | Yes        |
| A/B testing access       | No   | No  | Yes        |
| Prompt analytics         | Basic| Full| Full       |
| Model selection          | No   | Limited | Full   |
| Custom output formats    | No   | Yes | Yes        |

### Customization Validation

| Check                    | Validation Rule                       | Error Code           |
| ------------------------ | ------------------------------------- | -------------------- |
| Token limit              | Override + base < max_tokens          | OVERRIDE_TOO_LONG    |
| Safety compliance        | No prohibited content                 | SAFETY_VIOLATION     |
| Variable references      | All variables defined                 | UNDEFINED_VARIABLE   |
| Schema compatibility     | Output matches expected schema        | SCHEMA_MISMATCH      |

## Prompt Performance Metrics

### Quality Metrics

| Metric                  | Description                          | Target       |
| ----------------------- | ------------------------------------ | ------------ |
| Task Completion Rate    | % of tasks successfully completed    | > 90%        |
| Output Quality Score    | Human evaluation score (1-5)         | > 4.0        |
| Hallucination Rate      | % responses with factual errors      | < 2%         |
| Format Compliance       | % responses matching output schema   | > 98%        |

### Operational Metrics

| Metric                  | Description                          | Target       |
| ----------------------- | ------------------------------------ | ------------ |
| Latency (p50)           | Median response time                 | < 2s         |
| Latency (p99)           | 99th percentile response time        | < 10s        |
| Token Usage (avg)       | Average tokens per request           | < {{target}} |
| Cost per Request        | Average cost per prompt execution    | < ${{cost}}  |
| Error Rate              | % of failed prompt executions        | < 1%         |
